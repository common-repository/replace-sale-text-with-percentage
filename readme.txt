== Installation ==
1. Upload the folder "replace-percentage-badges-insted-of-sales-text" to "/wp-content/plugins/" path.
2. Activate the plugin through the "Plugins" menu in WordPress backend
3. After activating the plugin, Sale Text will automatically be replaced with Percentage.
4. Enjoy!
